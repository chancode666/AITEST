# THE AI LABYRINTH (AI 활용 능력 진단 게임)

## 1. 프로젝트 개요
이 프로젝트는 기존의 정적인 'AI 활용 능력 모의고사'를 **로그라이크(Roguelike) 스타일의 웹 게임**으로 재구성한 것입니다. 사용자는 AI 시대의 생존자가 되어, 각종 오개념(몬스터)을 처치하고 자신의 역량을 증명해야 합니다.

## 2. 주요 기능
- **게임화된 평가 시스템**: 
  - 정답 시: 몬스터 처치 (+점수, +MP 회복)
  - 오답 시: 플레이어 피격 (-HP 감소, 화면 흔들림 효과)
- **전략적 스킬 사용**:
  - **데이터 조회 (MP 30)**: 오답 1개 제거
  - **시스템 복구 (MP 40)**: HP 30 회복
  - **네트워크 우회 (MP 50)**: 해당 문제 패스
- **랜덤 스테이지**: 총 14개의 문제 은행 중 10문제가 무작위로 출제됩니다.
- **결과 리포트**: 게임 종료 후 등급(Rank)과 전투 로그를 통해 피드백을 제공합니다.

## 3. 파일 구조
- `index.html`: 게임의 메인 엔트리 포인트 (인트로 -> 전투 -> 결과 화면)
- `css/style.css`: 사이버펑크/레트로 퓨처 스타일의 디자인 스타일시트
- `js/data.js`: 문제 데이터 및 몬스터/스킬 메타데이터 정의
- `js/game.js`: 게임 상태 관리, 전투 로직, UI 업데이트 스크립트

## 4. 개발 및 확장 가이드
- **문제 추가**: `js/data.js`의 `GAME_DATA.questions` 배열에 새로운 문제를 추가하면 자동으로 반영됩니다.
- **밸런스 조정**: `js/game.js` 내 `STATE` 객체의 초기값이나 스킬 비용을 조정하여 난이도를 조절할 수 있습니다.
- **사운드 효과**: 현재는 플레이스홀더(`playSound`)로 되어 있으며, 실제 mp3 파일을 연결하여 타격감을 높일 수 있습니다.
